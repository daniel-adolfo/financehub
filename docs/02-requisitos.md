REQUISITOS FUNCIONAIS

RF001 - O sistema deve permitir o cadastro de usuários.

RF002 - O sistema deve permitir autenticação utilizando e-mail e senha.

RF003 - O sistema deve permitir cadastrar contas financeiras.

RF004 - O sistema deve permitir editar contas.

RF005 - O sistema deve permitir excluir contas sem movimentações.

RF006 - O sistema deve permitir cadastrar categorias.

...

REQUISITOS NÃO FUNCIONAIS

RNF001 - O sistema deve possuir interface responsiva.

RNF002 - As senhas devem ser armazenadas de forma criptografada.

RNF003 - O backend deverá ser desenvolvido em Java utilizando Spring Boot.

RNF004 - O banco de dados será PostgreSQL.

RNF005 - O sistema deverá utilizar autenticação JWT.

RNF006 - O tempo médio de resposta das requisições deverá ser inferior a 2 segundos em condições normais.
